# dotg.tech booking

### Переменные окружения
> Находятся в `./config/default.json` <br />

### Функции

- алгоритм кэширования списка поездов (Redis)
- создание очереди для бронирования (Rabbit)
- бронирование мест
- алгоритм поиска билетов на ближайшую дату

### Запуск

#### docker compose -f ./config/default.json
